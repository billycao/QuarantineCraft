Getting Started

  1. Download Java if you don't already have it.
  2. Buy Java Minecraft.
  3. Download and extract QuarantineCraft_win.zip
  4. Launch QuarantineCraft.

  7. Enjoy! Join our Discord for shenanigans.

"I am a scrub and only have a Mac."

(Yes, it's more work but I still typed it out for you. ❤️)

  1. Download Java if you don't already have it.
  2. Buy Java Minecraft.
  3. Download Minecraft.
    (Do the whole drag to Applications folder thing.)
  4. Download and install Forge.
    (You may have to right-click -> Open)
  5. Download and extract QuarantineCraft_mac.zip
  6. Launch Minecraft.
  7. Log in.
  8. Navigate to Installations -> New...
  9. Version: Select "release 1.15.2-forge-31.1.27"
  10. Game Directory: Navigate to the QuarantineCraft folder you extracted in Step 5.
  11. OPTIONAL but recommended for improved performance: Under Advanced Options -> JVM Arguments enter this pile of garbo:
    -Xmx4G -Xms256m -XX:+DisableExplicitGC -XX:+UseConcMarkSweepGC -XX:+UseParNewGC -XX:+UseNUMA -XX:+CMSParallelRemarkEnabled -XX:MaxTenuringThreshold=15 -XX:MaxGCPauseMillis=30 -XX:GCPauseIntervalMillis=150 -XX:+UseAdaptiveGCBoundary -XX:-UseGCOverheadLimit -XX:+UseBiasedLocking -XX:SurvivorRatio=8 -XX:TargetSurvivorRatio=90 -XX:MaxTenuringThreshold=15 -Dfml.ignorePatchDiscrepancies=true -Dfml.ignoreInvalidMinecraftCertificates=true -XX:+UseFastAccessorMethods -XX:+UseCompressedOops -XX:+OptimizeStringConcat -XX:+AggressiveOpts -XX:ReservedCodeCacheSize=2048m -XX:+UseCodeCacheFlushing -XX:SoftRefLRUPolicyMSPerMB=10000 -XX:ParallelGCThreads=10 -XX:PermSize=512m -Dfml.ignorePatchDiscrepancies=true -Dfml.ignoreInvalidMinecraftCertificates=true -Duser.language=en -Duser.country=US
  12. Click Create.
  13. Play the instance you created! (Multiplayer -> QuarantineCraft)
  14. Enjoy! Join our Discord for shenanigans.
    I (Billy) am Nomka on Discord. :)

  • Why Valhelsia 2?

    Vanilla (unmodded) Minecraft often leaves a lot to be desired and gets stale quickly.

    However, mod packs are often behind the latest versions and don't allow you to experience the latest features.

    Valhelsia combines both worlds: It's compatible with the current latest version of Minecraft (1.15.2), but also is a popular, well supported mod pack with lots of nifty features. If you prefer to play Vanilla, you're free to ignore all the mod features and just punch some trees. :)

    For the rest of you, Valhelsia seems like it has tons of neat stuff to discover! It's my first time playing it too so I'll be discovering with you all. :)
